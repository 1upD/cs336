Answers for the lab 3 exercises:

  • 3.1
    • Node.js is a JavaScript runtime environment that provides event-driven non-blocking I/O for web applications. Express is a web framework built on top of Node.js.
    • packages.json provides information about a Node.js application and also manages dependencies. NPM packages a Node.js app require to run can be found in packages.json and automatically installed.
  • 3.2
    • Static files are static in that they cannot change dynamically the way that a web page can. The files can only be changed if they are changed remotely and redownloaded when the page is refreshed.
    • The server continues to serve up hello world messages. When you access the port, it still serves the hello world message. The static files are accessed through the /static/ path.
